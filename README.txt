苍穹19_hover版 - 职业按钮悬停效果 + 界面文字优化

== 文件说明 ==
苍穹19_hover版.exe  → 可直接在Windows双击运行的exe（已包含所有修改）

苍穹19_hover源码/   → 本次修改后的完整项目源码（方便下次在此版本继续优化）

== 本次修改内容 ==
1. ui.py (draw_menu函数)
   - 三个职业按钮添加鼠标悬停检测
   - 悬停时边框变紫色 (80,50,120)，文字高亮 (220,200,255)
   - 悬停和键盘选中时文字都会变亮
   - 去掉 -- RPG -- 和版本号文字
   - 提示文字改为：
     第1行：鼠标悬停/点击 选择职业
     第2行：鼠标点击移动  ·  数字键释放技能  ·  点击按钮自动挂机

2. input.py (新增handle_menu_mouse方法)
   - 鼠标悬停某职业按钮时切换选中
   - 点击已选中按钮则进入游戏

3. main.py (MOUSEBUTTONDOWN路由)
   - 新增 game.s=="menu" 时调用 handle_menu_mouse

== 注意事项 ==
- 键盘操作（↑↓选择 + Enter确认）仍然保留，作为备用
- 悬停效果与对话框按钮（存档等）风格保持一致
- exe依赖的库已全部打包，无需安装Python/Pygame
